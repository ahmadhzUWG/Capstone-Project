Remove Employees to Group
Context:
			Admin is viewing groups
		Primary Flow:
			1. Admin selects a group
			2. Admin selects "Manage Group"
			3. System loads Group Management Page
			4. Admin removes employees to group
			5. Admin selects "Confirm" to save changes
		Alternate Flow :
			Admin cancels managing the group
			A1. Admin cancels managing the group
			A2. Resume primary flow step 1
